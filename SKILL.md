# Pose Analysis Skill

用 MediaPipe 分析训练视频姿态，并让 AI 给出改进建议。

## 使用方法

告诉我要分析的视频文件路径：
```
用姿态分析 skill 分析 ~/Videos/击剑训练.mp4
```

## 功能

1. **姿态提取** - 用 MediaPipe 提取人体 33 个关键点
2. **AI 分析** - 将关键点数据发给 AI 模型
3. **改进建议** - 返回动作评分和改进意见

## 配置

- 虚拟环境：`~/.venv-pose`
- 视频目录：`~/Videos/`（请在此目录存放训练视频）
- AI 模型：**Gemini 2.5 Pro**（已配置 API Key）

## 支持的运动类型

- 击剑
- 攀岩
- 体能训练

## 输出示例

```json
{
  "动作": "弓步冲刺",
  "评分": "7/10",
  "问题": [
    "前膝略微超过脚尖",
    "上身后仰角度偏大"
  ],
  "建议": [
    "重心再压低 5cm",
    "保持躯干更加挺直"
  ]
}
```
